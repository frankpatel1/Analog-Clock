# 🕒 Analog Clock

A simple and elegant Analog Clock built using **HTML**, **CSS**, and **JavaScript**. The application displays the current system time with smoothly animated clock hands, providing a realistic analog clock experience directly in the browser.

## 📖 Overview

This project demonstrates how to create a real-time analog clock using front-end web technologies. It utilizes JavaScript to calculate and update the positions of the hour, minute, and second hands based on the current time.

The project is ideal for beginners learning JavaScript DOM manipulation, CSS transformations, and time-based animations.

## ✨ Features

* Real-time clock synchronization
* Animated hour, minute, and second hands
* Responsive design
* Lightweight and fast-loading
* Pure HTML, CSS, and JavaScript implementation
* No external libraries or frameworks required
* Cross-browser compatibility
* Clean and modern UI

## 🛠️ Technologies Used

## 🚀 Getting Started

### Prerequisites

You only need a modern web browser:

* Google Chrome
* Mozilla Firefox
* Microsoft Edge
* Safari

### Installation

Clone the repository:

```bash
git clone https://github.com/frankpatel1/Analog-Clock.git
```

Navigate to the project directory:

```bash
cd Analog-Clock
```

Open the application:

```bash
Open index.html in your browser
```

## ▶️ Usage

1. Launch the application in your browser.
2. The clock automatically synchronizes with your device's current time.
3. Observe the movement of the:

   * Hour hand
   * Minute hand
   * Second hand
4. Resize the browser window to experience responsive behavior.

## 🎯 Learning Objectives

This project helps developers understand:

* JavaScript Date Object
* DOM manipulation
* CSS transforms and rotations
* Real-time updates using intervals
* Front-end project structure
* Responsive web design principles

## 🔧 How It Works

The application:

1. Retrieves the current system time.
2. Calculates the rotation angles for each clock hand.
3. Applies CSS transformations dynamically.
4. Updates the clock every second to maintain accurate time display.
